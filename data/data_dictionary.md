| Column Name | Explanation | Additional information |
| ----- | ---------- | ----- |
| id | Identifies univocally each entry of the dataset. Ideal for databases | none |
| title | Name of the electronic literature object | none |
| author_n_surname | First letter of the name followed by a dot and the surname | In case of multiple authors, they are comma separated |
| creator | characteristics of the creator(s) | ***person***: singular author; ***collaboration***: multiple authors; ***collective***: if there is a collective |
| publication_year | Year of first publication | If not yet published > Unpublished |
| url | link to the project metadata and access | none |
| access_date | last access to the project | none |
| original_journal | original journal publisher | none |
| type | Different genres of e-literature files seen in class | The possible entries are: ***hypertext fiction or hypertext poetry***, ***Interactive fiction and literary games*** ***Network fiction or social media works***, ***Kinetic or “Flash” poetry (platform specific)***, ***Codework and language code hybrids***, ***Generative or algorithmic writing***, ***Local narratives and installations*** |
| genre | Traditional narrative genre | none |
| technology | technology and/or software used for the creation of the elo | none |
| computer_languages | programming languages used for the creation and preservation | none |
| accessibility| explaining eventual hardware or network constraints | none |
| interaction_level | Levels of interaction from the user | ***Low*** if the interaction is basically non existent, and if the hypertextuality is either ephemeral or completely missing; ***Medium*** if either the user is relatively engaged in the content or if the hypertextuality produce different yet still number limited and never-changing choices; ***High*** if either the level of hypertextuality produced by the e-literature object creates complex structures in perpetual or very high chance of changing |
| sensory_experience | Physical senses involved by the e-literature object | none |
| textuality | types of textual object  | ***linear***: normal text, read from top to bottom; ***non-linear***: opposite to linear; ***modular***: text made of small, independent, reusable units |
| version | Version of the project | none |
| versioning | metadata about the version(s) | ***stable***: not modified; ***updated***: went through some updates; ***multiple versions***: there is more than one version available at the moment; ***unclear***: there are no informations |
| language | language of the elo | in this case, all are made in English |
| metadata_sources | source of the ELO | they all come from an ELO directory |
| preservation_status | Shows the preservation status of the project | ***Unavailable***: impossible to reach the resources; ***Undergoing preservation***: momentarily unavailable due to undergoing restoration practices; ***Partial***: only a section of the project is visible; ***Perfect***: available without restrictions, or if the restrictions are very limited |
| licence | type of licence | none |
| one_sentence_description | Short description of the project | max 200 characters |
